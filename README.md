# Views e Triggers em Banco de Dados

## Descrição do Projeto
Projeto prático focado na criação de views para personalização de acesso às
informações e triggers para controle de eventos em banco de dados.

---

## Parte 1 – Views e Permissões
Foram criadas views para facilitar consultas estratégicas e definidos perfis
de usuários com diferentes níveis de acesso às informações.

---

## Parte 2 – Triggers
Triggers foram implementadas para registrar remoções de usuários e controlar
atualizações de dados no cenário de e-commerce.
